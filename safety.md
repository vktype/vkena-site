# ✦ Safety Inquiries

## What is this space?

This is a community-driven archive created to preserve a directory of editing resources once the original source goes offline. Because the future of the original links is uncertain, these files have been mirrored onto private backup links to ensure they remain accessible to editors who rely on them.

## Credits & Origin

Please note that we are strictly **reuploaders**, not software developers or the creators of these packages. Just like the original founders at Satvrn, we do not modify, crack, or program these files. We are simply archiving a previously trusted and clean distribution hub so it isn't lost in time.

## Future Updates

Because this is a static archive, **these files will not be updated.** We will not be uploading future versions of these tools or software patches. What is currently in the directory is all that will be available.

## User Responsibility & Safety <span class="note2">(read before downloading)</span>

While the original distribution hub has been widely recognized as a trustworthy, safe staple within the editing community for a long time, downloading and setting up modified software always carries inherent risks.

- You are entirely responsible for your own device’s security.
- If you do not know how to safely manage file extractions, troubleshoot installation blocks, or do your own technical research, **do not proceed.** If you are uncomfortable taking these steps, you are highly encouraged to play it safe and officially purchase the services and software directly from the legitimate creators.
