# ✦ How To Extract
---
## ✧ Windows <span id="windows"></span>
Install [7-Zip](https://www.7-zip.org/) or alternative software. For windows users, we recommend [7-Zip](https://www.7-zip.org/) as it is both free and open source. Alternatives include NanaZip and WinRAR.

Click [here](https://files.catbox.moe/zpz8qt.mp4) for a video tutorial.
1. **Right click** the file you wish to open.
2. Select **7-Zip** *under **Show More Options** on windows 11*.
3. **Extract here** > Enter password > **OK** **(password:** `star`**)**
4. Proceed with your new, unzipped folder.

---
## ✧ MacOS <span id="macos"></span>
Install [Keka](https://www.keka.io) or alternative software. For Mac users, we recommend [Keka](https://www.keka.io) as it is both free, and open source. Alternatives include The Unarchiver and PeaZip.
<span class="note">Only the AppStore version is paid, while the [download](https://ibb.co/Wp2xtcnb) on their site is free.</span>

Click [here](https://files.catbox.moe/sf7gfp.mp4) for a video tutorial.
1. **Right click** the file you wish to open.
2. Select **Open With** > choose **Keka**.
3. Select location if asked > Enter password > **Done** **(password:** `star`**)**
4. Proceed with your new, unzipped folder.

<span class="note">Archive utility can open encryptyed files in .zip format, but not .7z. Files will vary.</span>
